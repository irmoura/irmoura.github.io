Projetos

  • GAX16

    O GAX16 surgiu com a idéia de organizar setores onde é necessário controlar a ordem de atendimento de cada colaborador do estabelecimento.
    Quando comecei a desenvolver o GAX16 tive a oportunidade de aperfeiçoar meus conhecimentos ao utilizar códigos que possibilitavam a sintetização de voz, uso de banco de dados MySQL em rede e a línguagem C#.

  • Barbearia Pinheiro

    Com o Barbearia Pinheiro a idéia foi criar um app de agendamento simples.
    Com o desenvolvimento do Barbearia Pinheiro aprendi muitas coisas que me ajudaram em vários momentos, como, uso de entidades, login externo usando o Facebook entre outros. O Barbearia Pinheiro utiliza o Firebase como banco de dados para guardar as informações de agendamento.
